## 🚀 What It Is
Sytrus for Windows is Image-Line's flagship modulation synthesizer, closely associated with FL Studio and loved for the depth it packs into a single window. The Sytrus FL Studio plugin loads as a VST synth in FL Studio and other compatible hosts, letting you combine six operators, ring modulation, and a subtractive stage to sculpt tones that would otherwise take several instruments. Whether you reach for the fl studio sytrus presets or build a patch from scratch, this Sytrus synth rewards curiosity with an almost limitless palette.

> Six operators, a full modulation matrix, and a built-in effects chain make Sytrus a complete sound-design environment rather than a single-trick synth.

## ✨ Key Features
- 🔀 **FM + ring modulation** — Route six operators through a flexible matrix to layer frequency modulation and ring modulation into complex, evolving timbres.
- 🎚️ **Subtractive engine** — Filters and oscillator shaping sit alongside the FM stage, so the Sytrus VST spans bright digital tones and smooth analog-style sounds.
- 🌊 **Deep modulation** — Envelopes, LFOs, and mapping curves connect to nearly every parameter for expressive, moving patches.
- 🎧 **Built-in effects** — An onboard rack with reverb, delay, chorus, and EQ finishes your patch inside the plugin.
- 🧩 **Host-friendly** — Runs as a plugin instrument in FL Studio and other compatible hosts, fitting neatly into any project.

## 🎯 First Steps
1. Add Sytrus to a track in FL Studio or another compatible host.
2. Audition the factory presets to get a feel for what the engine can do.
3. Tweak the operator matrix to change how the operators modulate one another.
4. Layer in LFOs and envelopes, then polish with the built-in effects.
5. Save the result as your own preset for future projects.

## ❓ Questions
**Is Sytrus free?**
Sytrus ships with certain FL Studio editions and is available as a paid add-on for others. The FL Studio trial is the honest way to try it before purchasing.

**Which versions of Windows are supported?**
Sytrus for Windows works with current 64-bit editions of Windows, such as Windows 10 and Windows 11, when running inside a supported host.

**Sytrus vs Serum — which should I pick?**
It depends on your style. Sytrus is an FM/RM and subtractive synth centered on operators, while Serum is a wavetable synth. Both sound excellent; choose the one whose workflow fits the sounds you want to make.
